# Luxe Booking – Opulent Odyssey

## Project Overview
Luxe Booking – Opulent Odyssey is a multi-page luxury hotel booking platform designed to deliver a premium user experience for discovering destinations, exploring hotel and adventure offerings, and managing bookings through a clean, modular front-end architecture.

## Tech Stack
- Vite
- Vanilla JavaScript (ES Modules)
- Bootstrap
- Supabase

## Pages Implemented So Far
- Home (`index.html`)
- Hotels
- Destinations
- Adventures
- Bookings
- Login
- Register
- User Page
- Admin

## Local Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Project Architecture (Short)
This project follows a **multi-page architecture** where each page has a dedicated HTML entry and corresponding JavaScript module. Shared UI elements (such as header and footer) are organized as reusable modules, while page-specific logic is kept in separate page scripts. The structure is intentionally modular to improve maintainability, readability, and scalability for future feature expansion.
